# Tree 树形控件

用清晰的层级结构展示信息，可展开或折叠。

依赖组件：`ElTree` 需先全局注册。
组件的属性和事件, 插槽和ElTree一致 [ElTree](https://element-plus.gitee.io/zh-CN/component/tree.html#%E5%B1%9E%E6%80%A7)
